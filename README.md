# Terminal-Zomboid
Terminal Zomboid adalah game survival teks bergaya retro yang menempatkan pemain di dunia pasca-apokaliptik penuh zombie. Jelajahi kota, kelola stamina, lapar, dan kesehatan, temukan loot acak, craft perlengkapan, serta bertahan dari serangan musuh. Semua aksi berlangsung lewat antarmuka terminal cepat, sederhana, dan intens.
